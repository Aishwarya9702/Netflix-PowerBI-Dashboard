# Netflix DashBoard - Power BI

## Overview
This project presents an interactive Netflix Dashboard built using Power BI. The dashboard provides insights into Netflix's content library, including content growth trends, ratings distribution, genre popularity, and country-wise availability. It also includes a title-level analysis page where users can explore detailed information about individual movies and TV shows.

## Tools Used
Power BI | MySQL | Excel/CSV Dataset

## Key Insights
- Netflix's content catalog has expanded significantly over the years.
- Mature audience content represents a substantial portion of the library.
- Drama and International Movies are among the leading genres.
- Netflix offers content from numerous countries, reflecting its global reach.
- Interactive title-level analysis enables deeper exploration of individual shows and movies.
